# Python Practice Exercises 🐍

This repository contains a curated collection of Python exercises I've completed as part of my learning journey toward becoming an AI/ML developer. Each exercise is organized by topic for clarity, readability, and continuous improvement.

## 📁 Folder Structure

Each folder focuses on a fundamental area of Python programming:

### `01_variables_input_output/`
Basic user input, data types, and string concatenation using `input()` and `print()`.

### `02_basic_math_operations/`
Covers arithmetic operations, sum and average calculations, and basic math applications.

### `03_conditionals_and_branching/`
Includes `if`, `elif`, `else` branching, logic checks, and user-driven condition handling.

### `04_loops_and_lists/`
Practice with `for` loops, list building, iteration, and processing collections of inputs.

### `05_strings_and_formatting/`
Explores string methods, formatting with f-strings, and manipulation using `.join()` and `.upper()`.
